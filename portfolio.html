HTML and CSS

HTML (Hyper Text Markup Language) and CSS (Cascading Style Sheets) are the foundational technologies for creating web pages. HTML provides the structure, while CSS defines the style and layout.These are the very first thing that I got to know. I used to think we just right coding tags out like an essay.

The Box Model

The CSS box model is a container that contains multiple properties including borders, margin, padding, and the content itself. It is used to create the design and layout of web pages. I love this lesson. It helps seperate and organize the content. The web looks so much more interesting and neat.

Website Navigation

The "nav" HTML element represents a section of a page whose purpose is to provide navigation links, either within the current webpage or to other webpages. This explain so much about how click the button or link and it brings you to a different page work. It amazes me and I really enjoy learning it.
